## utils.js

Този файл съдържа помощни (utility) функции,
които се използват на различни места в приложението.
Основната му цел е да изолира обща UI логика,
която не е пряко свързана с Firebase или бизнес логиката.

Функцията `showMessage` отговаря за показването
на кратки съобщения към потребителя,
като например потвърждение за успешно изпратена форма
или съобщение за грешка.

Функцията приема два параметъра:
– `text` – текстът, който ще бъде показан
– `type` – типът на съобщението (по подразбиране „success“)

Параметърът `type` се използва като CSS клас,
което позволява различен визуален стил
за успешни съобщения и съобщения за грешка.

Вътре във функцията първо се намира HTML елементът
с идентификатор `message-container`,
който служи като контейнер за всички съобщения.

След това текстът на контейнера се задава
на подадения текст,
а класът на елемента се променя според типа на съобщението.

Функцията използва `setTimeout`,
за да изчисти съобщението автоматично след 3 секунди.
Това става чрез премахване на текста и CSS класа,
което връща контейнера в първоначалното му състояние.

По този начин потребителят получава ясна обратна връзка,
без да е необходимо да затваря съобщения ръчно
или да използва стандартни browser alert-и.
